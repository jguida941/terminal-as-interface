# The Terminal as Interface

**AI CLI Tools and the New Programming Workflow**

This essay explores how AI command-line tools are reshaping software development — from how code is written and validated, to how programmers learn, collaborate, and think about authorship and labor.

It draws on firsthand experience building [VoiceTerm](https://github.com/jguida941/voiceterm), a Rust-based voice overlay for AI CLIs, as a primary case study.
