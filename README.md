# The Terminal as Interface

## AI CLI Tools and the New Programming Workflow

This project publishes a public overview, technical companion, and evidence
appendix on how AI command-line tools are reshaping software development.

It draws on firsthand experience building
[VoiceTerm](https://github.com/jguida941/voiceterm), a Rust-based voice overlay
for AI CLIs, as a primary case study.
